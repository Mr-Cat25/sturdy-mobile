# sturdy-mobile

A mobile application project.

## Getting Started

Clone the repository and follow the setup instructions below to get the project running locally.

## Prerequisites

- Node.js (v18 or later)
- npm or yarn

## Installation

```bash
npm install
```

## Running the App

```bash
npm start
```
